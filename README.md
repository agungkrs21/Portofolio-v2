# Agung Kurniawan — Portfolio

A modern developer portfolio built to showcase my projects, technical skills, and experience through an interactive and performant web experience.

The website highlights selected projects, recent GitHub activity, and my background as a Frontend Developer while focusing on clean design, smooth animations, accessibility, and performance.

## Features

- Modern and responsive interface
- Interactive landing page
- Project showcase with detailed case studies
- About section
- Skills and technology stack
- Recent GitHub commits
- Contact section
- SEO optimized
- Open Graph and structured metadata
- Dark mode support
- Smooth animations and page transitions

## Tech Stack

- Next.js
- React
- TypeScript
- CSS Modules
- Vercel

## Getting Started

Clone the repository.

```bash
git clone https://github.com/your-username/portfolio.git
```

Navigate to the project.

```bash
cd portfolio
```

Install dependencies.

```bash
pnpm install
```

Start the development server.

```bash
pnpm dev
```

Open your browser and visit:

```text
http://localhost:3000
```

## Build

Create a production build.

```bash
pnpm build
```

Run the production server.

```bash
pnpm start
```
